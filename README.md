## Intro.简介

EMM-MusicPlayer，一款自用的（自娱自乐的）本地音乐播放器。

用于体验/入门Electron开发，确实体验到了跨平台和开发上的方便，

以及 软件包臃肿不堪 和 运行效率低下 的Electron“优良传统”。
